Whitespace

As a review, we have learned that:
  1. A statement (System.out.println or etc..) must always end with a semicolon(;).
  2. Whitespaces or spaces can be used to indent our code, making it look neat.
  3. A text can be concatenated inside a System.out.print or System.out.println with the use of the addition operator (+) while enclosing the text with double quotation (" ") or single quotation (' ').
  4. The program must always have at least one class and only one public static void main.
  5. Each "{" must be partnered with a corresponding "}"
  6. Java is Case Sensitive.

After the double slash (//) ,there is text saying "Write your code here".

Those are called comments and it has no bearing within the program. It is used to help the programmer not only as pnemonic, but also as a guide. Comments can be inserted everywhere as long as it starts with a // followed by your desired text.
