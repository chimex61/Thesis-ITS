Introduction

One of the most popular programming languages is the Java programming language. It was originally developed by Sun Microsystems initiated by James Gosling and released in 1995 as core component of Sun Microsystems' Java platform.

So, the question is, why Java?

Ranging from software development, mobile applications, and large systems development, Java opens a number of opportunities for aspiring developers and experienced developers alike.

While these may seem daunting, it is also important to at least familiarize with some technical and boring stuff (You may skip these terms except for Object Oriented and Class):


Java makes an effort to eliminate error prone situations by emphasizing mainly on compile time(when code is converting to machine code), error checking (incorrect syntax), and runtime checking (when the program is running).

With Java's multithreaded feature it is possible to write programs that can perform many tasks simultaneously. This design feature allows the developers to construct interactive applications that can run smoothly.

Unlike programming languages including C and C++, when Java is compiled (converted to machine code),it is not compiled into platform specific machine, rather into platform independent byte code. This byte code is distributed over the web and interpreted by the Virtual Machine (JVM) on whichever platform it is being run on.

Java byte code is translated on the fly to native machine instructions and is not stored anywhere. The development process is more rapid and analytical since the linking is an incremental and light-weight process.

With the use of Just-In-Time compilers (when the program runs), Java enables high performance.

Java is designed for the distributed environment of the internet.

Java is Object Oriented Programming(OOP). Learning Java can enable you to master other OOP languages.

In Java, everything is an Object. For example in a real world scenario, a dog has states/characteristics - color, name, breed as well as behaviors - wagging the tail, barking, eating. Additionally, it is an instance of a class.

Human Ryan_Mabasa = new Human();

A Class can be defined as a template/blueprint that describes the behavior/state that the object of its type supports.
