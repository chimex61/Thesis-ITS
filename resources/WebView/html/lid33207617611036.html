Methods

Introduction

Did you ever wonder what public static void main(String[] args). It is actually a method responsible for running your other methods. A method is a collection of statements that are grouped together to perform an operation.

When you call System.out.println(), for example, the system actually executes a method consisting of several statements in order to display a message on the console.


A static method is a method that can be called on a class and usually does not require the class to be instantiated. You will learn what static is on the last module, but for now, always put static whenever you are declaring your methods.

A method must always be inside the class otherwise it will result to an error
