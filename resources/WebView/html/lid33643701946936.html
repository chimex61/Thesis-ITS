Object

We have already introduce what a Class is. Now you will learn to create your own object-oriented program. As an example, you have the main class Apples.


You can actually have other classes other than Apples, and use their properties or methods which you will learn later on. As you can see, betaMax is an instance of the class Radio and has all the properties or characteristics of a Radio.

The syntax for creating an object:

ClassName objectName = new ClassName()
