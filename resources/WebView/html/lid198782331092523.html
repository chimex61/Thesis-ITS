Review

As a review, we have learned that:
  1. Syntax for creating variables: datatype identifier = value
  2. Best practice in declaring a variable is a mixedCase. Interestingly, an identifier can begin with a currency character ($) or an underscore (_)
  3. char : single quotation ; String : double quotation
  4. int - used for integer values
    double - used for decimal values that ranges up to 15-16 decimal places
    char - used for a single character
    String - used for a series of characters
