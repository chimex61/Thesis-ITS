Module 3

Review

As a review, we have learned that:
  1. Syntax : Operand1 [insert operator here] Operand2
  2. Pre increment add the value by one then returns it.
  3. Post increment uses the value first before adding it by one. The same applies for decrement.
  4. The only difference of using pre increment and post increment is when you're printing them!
  5. The left variable or the variable on the left will always take the value on the right equation.
