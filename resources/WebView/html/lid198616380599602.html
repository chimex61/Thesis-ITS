System.out.println() adds a new line while a System.out.print does not. What's more, you can either use a double quotation (" ").

A text inside the System.out.println() or System.out.print should always start and enclosed with a " " to indicate that it is a text or specifically a String which you would learn later on.

Notice that class Apples and public static void main does not end with a semicolon (;) whereas the System.out.println() ends with it. The semicolon (;) indicates the signals the end of a statement such as the System.out.println() or System.out.print (click here).

Remember, each statement must always end with one semicolon (;)
