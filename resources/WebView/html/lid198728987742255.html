Module 2

Char and String

In this lesson, let us try to create variables with a text or a char inside.

As discussed in Lesson 1, a single quotation is reserved for a char data type whereas a double quotation is reserved for a String data type.

int myNumber = 69; (Best Practice)

char myCharacter = 'A';

String myText = "I can contain many characters!!!" //A String can contain many characters;
