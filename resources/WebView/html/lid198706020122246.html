Variable

The syntax for declaring a variable:

datatype identifier = value;

Here are some examples:

int myNumber = 69 (Best Practice);

double MyNumber = 3.14;

float mynumber = 3.14f; //always insert 'f' at every value of a float

As mentioned in the previous module, Java is Case Sensitive, in other words, myNumber and MyNumber are two different variables. In naming your variables, make sure that is simple, short, yet understandable. Do not repeat the same identifier/name. Additionally, A declaration of a variable is counted as a statement .

In setting the variable value, you do not use a single quotation (' ') or a double quotation (" ") unless it is a char or String data type. A single quotation is reserved for a char data type whereas a double quotation is reserved for a String data type.

But in a System.out.println or System.out.print, the use of a single quotation or double quotation is interchangeable. However, if you start with a single quotation, you must also end with a single quotation. The case is the same for the double quotation.
