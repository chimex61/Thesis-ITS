Module 4

Review

As a review, we have learned that:

if statement - consists of a boolean expression followed by one or more statements.

if...else statement - an if statement can be followed by an optional else statement, which executes when the boolean expression is false.

nested if statement - can use one if or if else statement inside another if or else if statement(s)...

switch statement - a switch statement allows a variable to be tested for equality against a list of value.

And (&&) - this checks if all conditions are satisfied. A single '&' is the same but less efficient.

Or (||) - this checks whether one condition is satisfied. A single '|' is the same but less efficient.

In an if else statement, there can only be a single if or else but there can be many else if statements.

The ternary operator (?) can be used to shorten an if else statement.

Conditions we commonly use:

checks if both values are equal

reverses the condition

checks if the variable is not equal to something

checks if the value is less than something

checks if the value is greater than something

checks if the value is less or equal than something

checks if the variable is greater than or equal than something
