Conditionals

In your Java Program, you can actually have a block of code executed first. We call this the if statement.

Basically, if the condition is true it will execute some code. Moving on, We actually have four cases of decision making statements:


consists of a boolean expression followed by one or more statements.

an if statement can be followed by an optional else statement, which executes when the boolean expression is false

you can use one if or if else statement inside another if or else if statement(s).

a switch statement allows a variable to be tested for equality against a list of value
