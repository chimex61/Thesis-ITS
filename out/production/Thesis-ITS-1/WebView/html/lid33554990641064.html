Objects and Classes

Object Oriented Programming(OOP) is a programming model where programmers focuses not only on the data structures, but also the type of operations(methods) that can be applied to the data structure.

  • Objects are the key to understanding object-oriented programming and the best way to understand how it works is by observing real-world objects.
  • A radio is an object. Its states would be on/off, current volume, current station, AM/FM, etc. and its behavior or functions that will make use or change the states would be turn on/off, increase volume, decrease volume, seek, scan, and tune.
  • Software objects are conceptually similar to real-world objects: they too consist of state and related behavior. An object stores its state in fields (variables) and exposes its behavior through methods.

Another important concept of OOP is the Class. In the real world, you'll often find many individual objects all of the same kind. There may be thousands of other radios in existence, all of the same make and model. Each radio was built from the same set of blueprints and therefore contains the same components.

In object-oriented terms, we say that your radio is an instance of the class of objects known as radios. A class is the blueprint from which individual objects are created. It can also be thought of as your implementation of the object
