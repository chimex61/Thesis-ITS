Basic Data Types

Introduction

A variable provides us with a named storage that our programs can manipulate. The names that we give to a variable are called identifiers.

All identifiers can begin with a letter (A to Z or a to z), a currency character ($) or an underscore (_).

And, each variable must have a specific data type which determines the size and layout of the variable's memory. It can either hold a range of data or a set of operations.
