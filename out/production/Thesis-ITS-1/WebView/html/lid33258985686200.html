Module 6

Parameter

As seen in the previous lesson, the void keyword allows us to create methods which do not return a value. Even though a method is declared void, it does not mean that it cannot have parameters.


Inside the parentheses ( ) are called parameters. It is responsible for taking inputs from your method. Apart from the void keyword, all methods with a data type will always need to return the same data type.

Note: a method declaration must always be in the class.
