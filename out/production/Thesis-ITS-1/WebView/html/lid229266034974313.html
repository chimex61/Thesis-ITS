Switch

Take a deep breath...

We still have more lessons coming up. Last but not the least the switch statement. It is similar to the if statement, but is more flexible when it comes to checking the expression or value.

Syntax:

Each case can have a number of statements. However, it must always end with the break statement in order to break out of the switch statement. All switch statements must have a default even if there is no statement to execute inside.
