Review

What we learned:

1. A method is a collection of statements that are grouped together to perform an operation. A method declaration must always be in the class.

2. Inside the parentheses ( ) are called parameters. It is responsible for taking inputs from your method.

3. The void keyword allows us to create methods which do not return a value. It can also have parameters

4. A static method is a method that can be called on a class and usually does not require the class to be instantiated. Always put static whenever you are declaring your methods.

5. All methods with a data type will always need to return the same data type

6.Aside from basic arithmetic operations, we can actually add conditionals and loops in our method.

7.When a value is passed into a parameter, that is called passing parameters by value.

8. When a class has two or more methods by the same name but different parameters, it is known as method overloading. It makes the program readable.

9. Local variables are variables created inside a method and cannot be accessed outside it.
Instance variables are variables created outside the class and can be accessed anywhere in the class.

10. You can actually use a method from another method. (click)
