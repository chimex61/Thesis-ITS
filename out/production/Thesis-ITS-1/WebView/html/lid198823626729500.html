Module 2

Initializing

In the beginning of the lesson the syntax for creating a variable:

datatype identifier = value; //this is actually called initializing.

Actually there is also another way in declaring a variable:

datatype identifier;

identifier = value;

int x;

x = 9; //this is done if you want to set the variable at a later time.

And another option of changing the variable's value:

int x = 11; //the value of x is 11.

x = 9; //the value of x is now 9.
