Polymorphism is generally, the ability to appear in many forms. In object-oriented programming, polymorphism refers to a programming language's ability to process objects differently depending on their data type or class. More specifically, it is the ability to redefine methods for derived classes

Another key concept in OOP is Inheritance. To understand this concept, observe the characteristics of a parent and child. The characteristics of the parent, beautiful eyes, short temper, peculiar head shapes, are inherited by the child and aside from these characteristics the child has other unique characteristics.

The parent class is referred as a superclass and the child class is referred as subclass. A superclass can have multiple subclasses(children).

Encapsulation is a fundamental concept in Object-Oriented Programming. It is a mechanism in Java where the variables and methods are wrapped together as a single unit. In encapsulation, the variables of a class will be hidden from other classes, and can be accessed only through the methods of their current class.

Think of it as a pill, the case is the encapsulation and the content are the data and methods.To encapsulate the data or hide the data, set the variables as private. When set as private, it can only be accessed through the methods of the class which are called setters and getters
